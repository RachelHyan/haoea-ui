## 使用包管理器

建议您使用包管理器 (NPM, <a href="https://classic.yarnpkg.com/lang/en/" target="blank">Yarn<h-icon name="link"></h-icon></a>, <a href="https://pnpm.io/" target="blank">pnpm<h-icon name="link"></h-icon></a>) 安装 HaoeaUI, 然后您就可以使用打包工具，例如 <a href="https://vitejs.dev/" target="blank">Vite<h-icon name="link"></h-icon></a> 和 <a href="https://webpack.js.org/" target="blank">webpack<h-icon name="link"></h-icon></a>。

```
# 选择一个你喜欢的包管理器

# NPM
npm install haoea-ui

# Yarn
yarn add haoea-ui

# pnpm
pnpm install haoea-ui
```
